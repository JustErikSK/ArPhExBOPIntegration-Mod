# ArPhEx: BOP Integration

ArPhEx: BOP Integration is a compatibility addon for **ArPhEx** and **Biomes O' Plenty**.

Its purpose is to extend the natural spawning of ArPhEx mobs into suitable Biomes O' Plenty biomes, making exploration feel more immersive with modded world generation.

Unlike replacing or overriding ArPhEx's spawning system, this addon simply adds additional biome entries. All original spawn mechanics including spawn rules, placement checks, and mob behavior remain completely unchanged.

## Configuration:

Every supported mob can be configured individually.

Available options include:

- Enable or disable spawning in modded biomes
- Edit the list of allowed modded biomes
- Change spawn weight
- Adjust minimum and maximum group sizes

## Requirements:

- Forge for 1.20.1
- ArPhEx mod (version 5.0 and above)

## Notes:

This addon only adds Biomes O' Plenty biome support by default, you can add biomes from different mods as well tho. Vanilla biome spawning continues to be handled entirely by ArPhEx.
